# 🧠 Resume Classification using Machine Learning

This project is an NLP-based machine learning system that automatically classifies resumes into different job categories using text processing and TF-IDF features.

## 🚀 Features
- Extract text from PDF and DOCX resumes
- Clean and preprocess text using NLP techniques
- Convert text into numerical features using TF-IDF
- Train multiple ML models for classification
- Compare model performance and select best model
- Save trained model for prediction

## 🤖 Models Used
- Logistic Regression (Best Performance)
- Naive Bayes
- SVM
- Random Forest

## 📊 Output
- Resume category prediction
- Model accuracy comparison
- Confusion matrix and evaluation metrics
